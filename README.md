#Tracker Pro

TrackerPro is a client-side personal finance dashboard built with vanilla HTML, CSS, and JavaScript — no frameworks, no backend.

Here's what it does:

tracks income and expenses with categories (food, transport, salary, freelance, etc.)
shows a live balance, total income/expense, and remaining budget
has a visual expense ratio bar that turns red when you're over 80% of income spent
lets you set a monthly budget and base salary, with a quick "Receive Salary" button
all data lives in localStorage — nothing leaves your device
supports searching/filtering transactions in real time
fully responsive with a slide-in sidebar on mobile
dark/light mode with preference saved across sessions
